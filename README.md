### 개선점

- Prerendering을 통해 각 경로 별 index.html을 만들기
- 좀 더 description, keyword, title을 페이지의 내용에 맞게 설정하기
- OG 이미지도 페이지에 사용된 이미지로 설정하기
- 좀 더 다양한 metadata를 사용
- 동적인 Favicon 사용

- 개선점에 있는 일부분에 대한 SEO 개선 후 스코어 분석
